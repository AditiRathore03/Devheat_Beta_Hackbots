# Quick Start 

📄 Clone or fork this repo :
`https://github.com/praharshsharma/MERN1.git`:

```sh
git clone https://github.com/AditiRathore03/MERN1.git
```

💿 Install all dependencies:

```sh
npm install
```

🚴‍♂️ Run your App:

```sh
cd backend
nodemon server.js

or

npm start

```
It runs on the browser with localhost 3000.
```sh
http://localhost:3000/
```
